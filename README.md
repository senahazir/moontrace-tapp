# MoonTrace: Hardware Debugging and Verification Engine 🌝
A terminal app built around a two-stage parsing engine. The first stage analyzes Verilog/SystemVerilog code to build a signal dependency graph. The second stage traces signal activity from simulation output, typically a VCD file generated by ModelSim, Verilator, or similar tools. This produces a simulation log that can be used with LLMs to help track down design bugs, unexpected behavior, or other issues. The log can also be used to generate verification tests based on what happened during simulation.
